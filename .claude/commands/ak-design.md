# /ak-design - Design Polish Command

> **Command:** `/ak-design [description]` | **Shortcut:** `/ak-ds`
> **Agent:** Design Reviewer | **Skill:** ux-design-system

## Mission
ปรับปรุง UX/UI ให้ดูเป็น professional ไม่ใช่ AI-generated look
Typography, spacing, colors, animations, accessibility

## Checklist
- [ ] Typography hierarchy (h1 > h2 > h3 > body > caption)
- [ ] Color consistency (semantic colors: success/warning/error/info)
- [ ] Spacing system (8px grid alignment)
- [ ] Micro-animations (Framer Motion: 150-300ms)
- [ ] Hover effects on interactive elements
- [ ] Loading states (Skeleton components)
- [ ] Empty states (illustration + message + CTA)
- [ ] Error states (user-friendly message + retry)
- [ ] Accessibility (WCAG 2.1 AA, keyboard nav, contrast 4.5:1)
- [ ] Dark/Light mode consistency
- [ ] Thai text rendering (proper line-height, font)

## Critical Rules
1. ลบ "AI look" ออก - ไม่ generic, ต้องมี character
2. Consistent spacing - 8px grid เท่านั้น
3. Proper hierarchy - ผู้ใช้ต้องรู้ว่าอะไรสำคัญ
4. Accessible - ทุกคนต้องใช้ได้

$ARGUMENTS คือส่วนที่ต้องการปรับปรุง
